# Tunnel-Modelling-generator-with-picture
사진과 IMU를 이용한 터널 모델링 및 결함,얼룩 위치 확인 

## Overview

터널 내부 주행 중 획득한 이미지와 IMU 데이터를 기반으로 터널 형상을 모델링하고,
AI 기반 결함 및 얼룩 탐지 결과를 3D 모델 상에 표시하는 시스템입니다.
